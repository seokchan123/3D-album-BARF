# 3D-album-BARF
오차 기반 적응형 주파수 스케줄러, 플로터 억제 정규화, 스케일 불변 깊이 가이드를 추가한 barf 사용 코드를 제공합니다.
기존 barf 모델을 다운로드 한 후 barf.py를 첨분한 enhanced_barf.barf.py로, llff.py를 enhanced_barf_llff.py로, nerf.py를 enhanced_barf_nerf.py로 교체한 후 run_barf.sh를 따라 실행하면 됩니다.

사용 데이터셋은 https://drive.google.com/file/d/1wWox8WW_0p9F3ILoJO3pIKDFeKf_na-3/view?usp=drive_link 에서 다운 가능합니다
